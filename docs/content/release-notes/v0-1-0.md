First release.

Three commands over the Dog CEO public API (no key needed):

- `dogapi breeds` - list all 100+ dog breeds sorted alphabetically, with sub-breeds comma-joined
- `dogapi random` - fetch one or more random dog images, optionally filtered by breed and sub-breed
- `dogapi images <breed>` - list images for a breed, with an optional sub-breed filter and client-side limit

The client paces requests at 100ms intervals and retries on 429 and 5xx responses (3 attempts by default).
